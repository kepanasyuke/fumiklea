# 🌌 Солнечная система — интерактивный 3D-симулятор

Этот проект представляет собой полностью автономный веб-приложение для визуализации Солнечной системы в реальном времени с использованием **Three.js**. В нём реализованы кеплеровские орбиты, реалистичные текстуры планет, динамические атмосферы, следы движения, а также встроены реальные изображения туманностей из космоса. Интерфейс выполнен в минималистичном футуристическом стиле с адаптацией под мобильные устройства.

> **Одна страница — вся мощь Three.js.**  
> Используются: WebGL, пост-эффекты, шейдеры, CSS2D-метки, система управления камерой и гибкие настройки качества.

---

## ✨ Особенности

- **Кеплеровы орбиты** — 8 планет, 5 карликовых планет, 8 спутников и 4 кометы движутся по эллиптическим траекториям с учётом эксцентриситета и наклонения.
- **Реалистичная визуализация планет** — процедурные шейдеры с шумом Перлина, создающие текстуры с деталями (полосы у газовых гигантов, кратеры, цветовые вариации).
- **Атмосферы и свечение** — для Земли и Венеры добавлены многослойные атмосферы с эффектом свечения на границе (рим-лайтинг), а также облачные слои с анимацией.
- **Следы (трейлы)** — визуальные линии, показывающие путь планет за последние N шагов.
- **Реальные туманности** — загружаются 8 реальных изображений туманностей (M42, Розетка, Орёл, Киля, Трёхраздельная, Лагуна, Конус, Пламя) с Wikimedia Commons и размещаются в 3D-пространстве.
- **Глубокое звёздное поле** — многослойная система фоновых звёзд и процедурные туманности (опционально).
- **Управление камерой** — вращение, масштабирование, сброс в исходную позицию. Также есть автоматический обзор (автовращение).
- **Панель управления** — скорость симуляции, выбор цели для фокусировки, включение/отключение следов, пауза, чувствительность зума.
- **Настройка качества** — 4 предустановки: «Производительность», «Сбалансированное», «Качественное», «Кинематографическое».
- **Dat.GUI** — дополнительная панель с тонкими настройками (сила Bloom, порог, масштаб времени, логический буфер глубины и др.).
- **Адаптивный дизайн** — интерфейс подстраивается под экраны шириной менее 800px.

## 🖥️ Технологии

- **Three.js (r128)** — рендеринг 3D-сцены.
- **OrbitControls** — управление камерой.
- **EffectComposer + RenderPass + UnrealBloomPass** — пост-эффекты (свечение).
- **CSS2DRenderer** — отображение текстовых меток созвездий.
- **Lil-gui** — панель расширенных настроек (опционально).
- **Шейдеры GLSL** — для динамических материалов планет и атмосфер.
- **Canvas API** — генерация текстур туманностей (процедурных) и колец Сатурна.
- **Fetch API** — загрузка реальных изображений туманностей и звездных каталогов (HYG).

## 🚀 Запуск

1. **Скачайте** файл `index.html` (это единственный файл проекта).
2. **Откройте** его в современном браузере, поддерживающем ES-модули (Chrome, Firefox, Safari, Edge).
   - Для локальной работы рекомендуется использовать **Live Server** (VS Code) или любой другой статический сервер (например, `python -m http.server`), чтобы избежать CORS-проблем при загрузке внешних изображений.
3. При первом запуске могут загружаться изображения туманностей и звездный каталог HYG — дождитесь появления надписи «Сцена готова» в нижнем левом углу.

> **Примечание:** Все зависимости подгружаются через importmap из CDN (unpkg). Интернет-соединение обязательно.

## 🎮 Управление

| Действие | Способ |
|----------|--------|
| Вращение камеры | **Зажать левую кнопку мыши** и двигать курсор |
| Приближение/отдаление | **Колесо мыши** или жест двумя пальцами на тачпаде |
| Выбор объекта для фокусировки | Выпадающий список «ЦЕЛЬ» в панели управления |
| Сброс камеры | Кнопка **«СБРОС КАМЕРЫ»** |
| Пауза / продолжение | Кнопка **«ПАУЗА»** или клавиша **Пробел** |
| Включение автовращения камеры | Кнопка **«АВТООБЗОР»** |
| Регулировка скорости симуляции | Кнопки **«−» / «+»**, сброс через **«СБРОС»** |
| Изменение чувствительности зума | Слайдер **«ЧУВСТВ. ЗУМА»** |
| Включение/выключение следов планет | Кнопка **«СЛЕДЫ»** |
| Быстрый переключатель качества | Клавиши **1** (Производительность) … **4** (Кинематографическое) |
| Сброс всей симуляции | Кнопка **«СБРОС СИМУЛЯЦИИ»** |

## ⚙️ Настройки (Dat.GUI)

При желании вы можете открыть панель **«Настройки»** в правом верхнем углу и изменить:

- **Качество** — предустановки производительности.
- **Сила свечения** — интенсивность Bloom-эффекта.
- **Порог свечения** — порог яркости для Bloom.
- **Следы планет** — включение/выключение трейлов.
- **Орбиты планет** — отображение орбитальных линий.
- **Масштаб времени** — скорость симуляции (0–4x).
- **Лог. буфер глубины** — включает логарифмический буфер (может помочь при очень больших расстояниях, но требует поддержки WebGL).
- **Динамическое начало координат** — экспериментально.
- **Названия созвездий** — отображение текстовых меток.
- **Реальные звёзды (HYG)** — загрузка каталога HYG для реалистичного звёздного фона (вкл/выкл).
- **Галактики (OpenNGC)** — загрузка галактик из каталога (в текущей версии отключено по умолчанию, чтобы не перегружать сцену).

## 📁 Структура

Проект представлен **одним HTML-файлом**, в котором объединены:

- **CSS** – стили интерфейса (темный фон, стеклянный эффект, акцентные цвета).
- **HTML** – разметка информационной панели, элементов управления и загрузочного сообщения.
- **JavaScript (ES Module)** – весь код симуляции: 
  - настройки, данные орбит, классы небесных тел,
  - инициализация сцены, камеры, рендереров,
  - создание планет, спутников, комет, астероидов,
  - загрузка звёзд и туманностей,
  - анимационный цикл, управление,
  - интеграция с Dat.GUI.

Все внешние библиотеки подгружаются из CDN через `importmap`.
