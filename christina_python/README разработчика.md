# Проект "Кристина" (По мотивам Стивена Кинга)

Пиксельный процедурный генератор сцен на базе Python и NumPy, вдохновленный культовым романом Стивена Кинга «Кристина» и хоррором «ОНО». Скрипт рендерит кинематографичные сцены из жизни одержимого автомобиля Plymouth Fury 1958 года.

## Текущий статус разработки (откатила до only python редактирую на локальном компьютере)
* [x] **Сцена 8:** Вид из салона автомобиля. Симулирует ослепление дальним светом от встречных фар Кристины. Реализована динамическая работа дворников с булевой маской теней, преломление света на каплях дождя и бирюзовая подсветка приборов.
* [x] **Сцена 9:** Ночное шоссе. Вид из кабины, летящей на высокой скорости. Реализована фиксированная двухпиксельная разметка в движении, конус света фар и жуткий детализированный силуэт человека в желтом капюшоне (Георги/Пеннивайз) с анимированным красным воздушным шаром на обочине.

## Системные требования
* Python 3.10 или выше
* Библиотека `numpy`


# 🚗 Кристина: Одержимый Ретро-Квест (32x32 LED Matrix Edition)

Добро пожаловать в текстовый интерактивный хоррор-квест по мотивам культового романа Стивена Кинга **«Кристина»**! Проект сочетает в себе логику классического текстового приключения и покадровую графику, стилизованную под светодиодный экран **Sega 32x32**.

---

## 🎮 Особенности проекта

* **Двухпоточный движок (Python + JS):** Python обсчитывает покадровую анимацию, физику и параметры, а веб-интерфейс HTML5/CSS3 рендерит картинку со скоростью 25 FPS с аппаратным ускорением GPU.
* **Ретро-атмосфера CRT:** Экран снабжён фильтром сканирующих линий (Scanlines) для имитации старых ЭЛТ-мониторов игровых автоматов 80-х годов.
* **Ультра-детализированный пиксель-арт:** Кастомные сцены (регенерация капота, взрывы, интерьер салона махагони, Пеннивайз в дождевике на обочине и гигантский знак STOP с анатомическими человеческими глазами).
* **Интерактивный терминал Sega:** Управление квестом и общение с Кристиной происходит через встроенную командную строку.

---

## 🛠️ Инструкция по установке и запуску

Игра тестировалась и работает на связке **Windows + WSL (Ubuntu)**.

### 1. Подготовка папки
Убедитесь, что файлы проекта лежат в одной директории:
```bash
~/christina_python/
  ├── christina.py   # Основной движок игры на Python
  └── index.html     # Веб-интерфейс квеста (Sega Terminal)
```

### 2. Освобождение портов (при необходимости)
Если порт `8000` занят прошлыми процессами, очистите его одной командой:
```bash
kill -9 \$(lsof -t -i:8000)
```

### 3. Запуск игры
Запустите графический сервер Python через консоль WSL:
```bash
python3 christina.py
```
Движок автоматически скомпилирует все 19 сцен анимации, поднимете локальный API-сервер и **самостоятельно откроет браузер** на странице игры `http://localhost:8000/index.html`.

## ⌨️ Команды внутриигрового терминала

Вводите эти команды в строку `C:\>` для взаимодействия с одержимым автомобилем:


| Команда | Описание действия | Ответ Кристины / Эффект |
| :--- | :--- | :--- |
| `help` / `помощь` | Выводит список доступных параметров | Показывает справочную системную информацию |
| `привет` | Первичное приветствие Кристины | Автомобиль зловеще подмигивает фарами в лог |
| `осмотр` | Анализ технического состояния ходовой | Кристина выдаёт отчёт о подвеске и амортизаторах |
| `ремонт` | Восстановление кузова (ФОРМА №1) | Анимированный статус-бар `РЕМ` поднимается до 100% |
| `гонка` | Переход в агрессивный режим охоты | Мотор V8 взрывается яростью, покрышки дымятся |
