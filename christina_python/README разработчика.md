Пиксельный процедурный генератор сцен на базе Python и NumPy, вдохновленный культовым романом Стивена Кинга «Кристина» и хоррором «ОНО». Скрипт рендерит кинематографичные сцены из жизни одержимого автомобиля Plymouth Fury 1958 года.

### 🟥 Блок восстановления и гаража (Сцены 0–5)
* **Сцена 0: Одержимый огонь (Помятая Форма №1)** — Кристина объята ядовитым пламенем, глитчующие искры и оранжевые всполохи V8 пробиваются сквозь обугленный кузов.
* **Сцена 1: Регенерация бампера (Show Me)** — Легендарная сцена из фильма. Смятый хромированный бампер плавно выпрямляется, металл вправляется на место, а по готовому крылу пробегает ослепительный белый блик.
* **Сцена 2: Одержимый Спидометр** — Кроваво-красная стрелка спидометра бешено скачет под мертвенно-зеленым неоном шкал.
* **Сцена 3: Выхлоп и крылья Кристины** — Тяжелая корма автомобиля крупным планом. Стоп-сигналы зловеще пульсируют рубиновым светом, а из хромированных труб густыми клубами валит белый дым.
* **Сцена 4: Преследование Бадди** — Экстремальный свет в лицо. Два огромных конуса дальнего света фар Кристины стремительно расширяются, ослепляя жертву и заливая экран белой вспышкой.
* **Сцена 5: Мастерская Дарнелла** — Неоновая вывеска зловеще мерцает и гаснет, пока по лобовому стеклу стекают синие капли дождя, а хром бампера ловит фиолетовые блики.

### 🛣️ Блок ночной охоты и саспенса (Сцены 6–10)
* **Сцена 6: Взрыв заправки** — Кульминация ярости Кристины. Огромный огненный гриб взрыва разрывает экран, пиксели плавятся от белого к оранжевому и желтому.
* **Сцена 7: Кровавый Титр** — Из пепла и затухающих угольков взрыва плавно рождается и грозно пульсирует неоновое слово `CHRISTINE`.
* **Сцена 8: Наклонные дворники и блики** — Вид изнутри салона. Две черные щетки дворников ходят наискосок по веерной дуге, перекрывая капли ливня, а по стеклу скользят блики ночных фонарей.
* **Сцена 9: Ночное шоссе (Пеннивайз и Взгляд из машины)** — Самая детализированная хоррор-сцена. Тёплый свет фар плавно нарастает к капоту, на правой обочине стоит Пеннивайз в жёлтом дождевике, чьи красные шары улетают точкой в верхний угол. В финале сцена переключается на гигантский центрированный знак `STOP` с белым окаймлением и облегчённой буквой `T`, под которым **зловеще закрываются живые анатомические глаза**.
* **Сцена 10: Одержимая деревянная Магнитола** — Крупный план центральной консоли. Текстура лакированного дерева махагони, решетка радиодинамика из теплого бакелита, кроваво-красная стрелка тюнера и **вылетающие из прибора объемные золотые ноты рок-н-ролла 50-х**.

* [ ] **Сцена 11:** Взгляд хищника. Панорамный вид спереди сквозь решётку радиатора Кристины. Симулирует преследование синего седана Бадди в упор. Реализовано ослепление цели дальним светом фар и динамическое отражение сминающейся кормы врага на хромированных клыках бампера Плимута.
* [ ] **Сцена 12:** Металлическая каша (Таран). Сцена столкновения бамперов «в упор». Реализована глитч-деформация синей пиксельной матрицы кузова Бадди, сжатие геометрии по вертикали и математический сноп радиально разлетающихся жёлто-оранжевых искр из эпицентра удара (y=16, x=16).

* [ ] **Сцена 13:** Кувырок и пламя. Финал погони. Реализована физика вращения повреждённого синего обломка машины вокруг своей оси по матрице поворота (cos/sin) с его одновременным улетом в левый верхний угол и последующей ослепительной ядерно-белой вспышкой взрыва.
* [ ] **Сцена 14:** Отрезвление (Вид из салона Арни). Возврат в кабину Кристины после убийства. Реализовано плавное угасание ядовито-зелёного свечения приборов до тусклого тёплого тона, капли ливня на стекле замедляются, а на лобовом стекле сквозь тьму проступают багровые разводы.
* [ ] **Сцена 15:** Ночной патруль. Появление полиции на шоссе. Реализован цикличный триггер красно-синей люстры копов, который каждые 4 кадра полностью окрашивает матрицу то точечным ультрамарином, то рубином, заставляя силуэты придорожных деревьев отбрасывать бегающие тени.
* [ ] **Сцена 16:** Окружение (Погоня копов). Вид сзади на Кристину, зажатую в коробочку полицией. Реализовано масштабирование приближающихся белых фар преследователей по бокам экрана и мигающие отблески сирен на глянцево-красном багажнике и острых плавниках кузова Кристины.
* [ ] **Сцена 17:** Прыжок через мост. Кульминационный побег. Камера зафиксирована сбоку. Реализована плавная параболическая траектория полёта красного прямоугольника Кристины над пиксельной сеткой тёмно-синей реки и вспыхивающие отрывные искры из-под заднего моста при толчке.
* [ ] **Сцена 18:** Финальный саспенс (Глаза во тьме). Экран полностью залит глухой ночной тьмой. Реализована попиксельная миндалевидная анатомия человеческих глаз Кристины в самом низу матрицы (строки 24-32). Взгляд с двойным белым бликом влажности зловеще подмигивает игроку и на кадрах 45-49 наглухо закрывается, завершая квест.

# 🚗 Кристина: Одержимый Ретро-Квест (32x32 LED Matrix Edition)

Добро пожаловать в текстовый интерактивный хоррор-квест по мотивам культового романа Стивена Кинга **«Кристина»**! Проект сочетает в себе логику классического текстового приключения и покадровую графику, стилизованную под светодиодный экран **Sega 32x32**.

---

## 🎮 Особенности проекта

* **Двухпоточный движок (Python + JS):** Python обсчитывает покадровую анимацию, физику и параметры, а веб-интерфейс HTML5/CSS3 рендерит картинку со скоростью 25 FPS с аппаратным ускорением GPU.
* **Ретро-атмосфера CRT:** Экран снабжён фильтром сканирующих линий (Scanlines) для имитации старых ЭЛТ-мониторов игровых автоматов 80-х годов.
* **Ультра-детализированный пиксель-арт:** Кастомные сцены (регенерация капота, взрывы, интерьер салона махагони, Пеннивайз в дождевике на обочине и гигантский знак STOP с анатомическими человеческими глазами).
* **Интерактивный терминал Sega:** Управление квестом и общение с Кристиной происходит через встроенную командную строку.

## 🛠️ Инструкция по установке и запуску

Игра тестировалась и работает на связке **Windows + WSL (Ubuntu)**.

### 1. Подготовка папки
Убедитесь, что файлы проекта лежат в одной директории:
```bash
~/christina_python/
  ├── christina.py   # Основной движок игры на Python
  └── index.html     # Веб-интерфейс квеста (Sega Terminal)
```

### 2. Освобождение портов (при необходимости)
Если порт `8000` занят прошлыми процессами, очистите его одной командой:
```bash
kill -9 \$(lsof -t -i:8000)
```

### 3. Запуск игры
Запустите графический сервер Python через консоль WSL:
```bash
python3 christina.py
```
Движок автоматически скомпилирует все 19 сцен анимации, поднимете локальный API-сервер и **самостоятельно откроет браузер** на странице игры `http://localhost:8000/index.html`.

## ⌨️ Команды внутриигрового терминала

Вводите эти команды в строку `C:\>` для взаимодействия с одержимым автомобилем:


| Команда | Описание действия | Ответ Кристины / Эффект |
| :--- | :--- | :--- |
| `help` / `помощь` | Выводит список доступных параметров | Показывает справочную системную информацию |
| `привет` | Первичное приветствие Кристины | Автомобиль зловеще подмигивает фарами в лог |
| `осмотр` | Анализ технического состояния ходовой | Кристина выдаёт отчёт о подвеске и амортизаторах |
| `ремонт` | Восстановление кузова (ФОРМА №1) | Анимированный статус-бар `РЕМ` поднимается до 100% |
| `гонка` | Переход в агрессивный режим охоты | Мотор V8 взрывается яростью, покрышки дымятся |
