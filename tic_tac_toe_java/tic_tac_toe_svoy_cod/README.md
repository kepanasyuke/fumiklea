## 📝 О проекте

Это классическая игра **«Крестики-нолики»** с графическим интерфейсом, написанная на **Java** с использованием **JavaFX** и **Maven**.  
Проект создавался в среде **IntelliJ IDEA** — от первой строки кода до последнего теста.

Игра предназначена для двух игроков на одном устройстве. Вы по очереди ставите X и O на поле 3×3.  
Побеждает тот, кто первым выстроит три своих знака в ряд (по горизонтали, вертикали или диагонали).

**Что умеет игра:**
- 🎮 Игра для двух игроков (человек против человека).
- 🖥️ Чистый и понятный интерфейс на JavaFX.
- 🧠 Автоматическая проверка победы и ничьей.
- 💬 Всплывающие уведомления с результатом партии.
- 🚫 Блокировка поля после окончания игры.

## 💻 Среда разработки

Проект полностью сделан в **IntelliJ IDEA** (Community Edition) с использованием **JavaFX** и Maven.

**Как всё собиралось:**
1. Создан новый проект через `File → New → Project → JavaFX`.
2. Подключён Maven для удобного управления зависимостями (вместо ручного добавления JAR-файлов).
3. В `pom.xml` прописаны зависимости:
   - `javafx-controls`
   - `javafx-fxml`
4. Интерфейс нарисован с помощью **Scene Builder** (встроенного в IDEA или отдельного приложения).
5. Класс `HelloController` реализует всю игровую логику: ходы, проверку победы, ничью и вывод сообщений.
6. Запуск происходит через кнопку `Run` или командой Maven: `javafx:run`.

**Почему IntelliJ IDEA + JavaFX?**  
Это современный и надёжный стек для десктопных приложений на Java. Он даёт быстрый старт, удобный отладчик и визуальный редактор интерфейса.

## 🛠️ Технологии и требования

| Компонент | Версия |
|-----------|--------|
| Java | 9+ |
| JavaFX | 11.0.2 |
| Maven | 3.6+ |
| IntelliJ IDEA | 2020.3+ (любая версия) |

**Основные зависимости (из `pom.xml`):**
- `javafx-controls` – элементы управления (кнопки, панели).
- `javafx-fxml` – загрузка интерфейса из FXML.
- `controls-fx` – дополнительные компоненты (опционально).
