## 📜 Сюжет

Сентябрь 1947 года. Гарвардский университет, вычислительная машина Mark II.  
Операторы замечают сбой в реле №70. При осмотре обнаруживается… **живая моль**, замкнувшая контакты.  
Ваша задача — аккуратно извлечь насекомое с помощью латунного пинцета и поместить в «Cooper Log Entry Area».  
Как только моль окажется в журнале, вы увидите историческую запись — первую в мире документальную фиксацию «бага» в компьютерной системе.

---

## ✨ Особенности

- 🎛️ **Эмуляция ЭЛТ‑монитора**  
  Сканирующие линии, фосфорное послесвечение, динамическая виньетка, случайные «царапины» и мерцание люминофора.

- 🐛 **Интерактивная моль**  
  Насекомое хаотично движется внутри реле, машет крыльями и «вибрирует» — поймайте его пинцетом.

- 📓 **Журнал событий**  
  Страница из отчёта 1947 года с эффектом печатной машинки и вклеенным «трофеем».

- 🔊 **Аутентичные звуки**  
  Звуки реле, щелчки пинцета, «жужжание» моли и старый принтер.

- 🖱️ **Полностью мышка**  
  Пинцет следует за курсором, клик по моли — поимка, клик по журналу — переход.

- 🖼️ **Визуальные глитчи**  
  Искажение текста, «дрожание» линий, эффект подгоревшего экрана.

---

## 🎮 Управление

| Действие                | Управление                          |
|-------------------------|-------------------------------------|
| Запуск демо / переход   | Левый клик мыши                     |
| Движение пинцета        | Перемещение мыши                    |
| Поимка моли             | Клик по моли (когда пинцет на ней)  |
| Продолжение после лога  | Левый клик в любом месте            |

> Подсказка: моль должна коснуться **зелёной рамки журнала**, чтобы запись завершилась.
