Этот проект — простой, но мощный **веб-калькулятор**, который умеет считать не только простые примеры вроде `2+2`, но и целые математические выражения с учётом скобок, приоритетов операторов и даже некоторых встроенных функций. Код очень компактный, но функциональный — в нём всего около 30 строк логики, включая обработку ошибок и историю вычислений.

### 🧠 Почему simpleeval?

В нём используется `simpleeval` — безопасная альтернатива встроенной функции `eval()`, которая никогда не применяется к пользовательскому вводу из-за высоких рисков. `simpleeval` позволяет выполнять только математические вычисления, игнорируя потенциально опасные команды. Для дополнительной безопасности можно также настроить список разрешённых функций и операций, что делает приложение устойчивым к вредоносным вводам.

Проект интересен тем, что показывает, как с помощью Flask можно создать полноценное веб-приложение с хранением истории в сессиях и гибкой логикой расчёта.

## 📝 О проекте

Простое, но функциональное веб-приложение — калькулятор, созданный с помощью микрофреймворка **Flask** на **Python**. Позволяет вычислять математические выражения любой сложности, поддерживает приоритет операций и скобки.

**Главная особенность** — вместо опасной встроенной функции `eval()` используется библиотека `simpleeval`, которая выполняет только математические вычисления, полностью игнорируя потенциально опасный код. Для дополнительной защиты вы можете настроить список разрешённых функций и операций прямо в коде, сделав приложение устойчивым к вредоносным запросам.

## 💡 Возможности

- 🧮 **Вычисление любых математических выражений** (поддерживаются `+`, `-`, `*`, `/`, `//`, `**`, `%`, скобки и приоритет операций).
- 📜 **История последних 12 вычислений** — сохраняется в сессии браузера даже после обновления страницы.
- 🧹 **Очистка истории** одной кнопкой.

- 🎨 **Минималистичный веб-интерфейс** с использованием HTML и CSS (шаблоны находятся в папке `templates`, статические файлы — в `static`).

## 🔧 Детали реализации

### Логика приложения (app.py)

Приложение использует **Flask** и хранит состояние в **серверной сессии**:

- **`session['history']`** — список последних до 12 вычислений (формат: `"выражение = результат"`).
- **`session['expression']`** — последнее введённое выражение (для повторного отображения в форме).
- **`session['result']`** — последний вычисленный результат.

#### Основные маршруты:

| Маршрут | Метод | Описание |
|---------|-------|----------|
| `/` | `GET` | Отображает форму калькулятора, историю и последний результат. |
| `/` | `POST` | Получает выражение из формы, вычисляет его и обновляет сессию. |
| `/history/clear` | `POST` | Очищает историю вычислений. |

#### Безопасные вычисления:

Вместо стандартной `eval()` используется **`simple_eval()`** из библиотеки `simpleeval`:

```python
from simpleeval import simple_eval
res = simple_eval(expr)   # expr — строка от пользователя
```

### Шаблоны и статика

- **`templates/index.html`** — основной интерфейс: форма ввода, отображение результата, кнопка очистки истории и список предыдущих вычислений.
- **`static/style.css`** — кастомные стили для аккуратного отображения.
