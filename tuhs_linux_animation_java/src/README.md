# 💾 FreaXExplorer v1.91 — Ретро-терминал археологии Unix & Linux


Элегантное десктопное приложение на **Java Swing**, стилизованное под монохромный зеленый CRT-монитор (люминофор P1, IBM 5151). Программа имитирует низкоуровневую «археологическую» выгрузку исторических артефактов Unix/Linux из архивов **TUHS (The Unix Heritage Society)**.

---

## 🛠️ Основные фичи

* **Симуляция дискеты 3.5"**: Пошаговое чтение магнитных дорожек (CHS-адресация) с аналоговыми задержками и случайными сбоями секторов (CRC Error).
* **Электронный PC Speaker**: Интеграция звукового отклика (`Toolkit.beep()`) при сбоях чтения и подтверждении операций.
* **Исторические пасхалки**: Реальные комментарии Линуса Торвальдса из исходников ядер `v0.01-0.11` и хроники дебатов с Эндрю Таненбаумом.
* **Глобальный Key Dispatcher**: Железобетонный перехват горячих клавиш, полностью защищенный от блокировок текстового поля.

---

## 🎮 Управление (Горячие клавиши)

* **`[F5]`** — Запустить механическую эмуляцию чтения дискеты с архивом.
* **`[F8]`** — Открыть оригинальный веб-сайт сообщества **TUHS** в браузере.
* **`[Y / N]`** — Интерактивный ответ на запрос перенаправления в конце симуляции.
* **`f-r-e-a-x`** — Секретный пасхальный код. Наберите его посимвольно в любой момент, чтобы вызвать покадровую ASCII-анимацию «моргающего» пингвина **Tux** и узнать историю кодового имени ОС.

🛑 Почему Makefile в Codespaces сейчас бесполезен?

Отсутствие дисплея: Java Swing была создана для домашних ПК. На серверах GitHub нет видеокарты и монитора, поэтому Java пытается подключиться к «воздуху».Блокировки туннелей: Утилита localtunnel часто режет веб-сокеты и бинарные потоки noVNC. Вместо окна программы вы видите белый экран, ошибку 404 или список папок vnc.html.Зачем усложнять? Чтобы запустить Java Swing в браузере «через дивы на JS», пришлось бы переписывать весь графический интерфейс с нуля на веб-технологии. Но есть готовый код на Java, который идеально приспособлен для запуска на вашем собственном компьютере!

🚀 Вариант 1 (Идеальный): Запуск в IntelliJ IDEA (в 1 клик)Это самый надежный способ. 

Всё, что вам нужно — перенести текстовый файл с кодом на ваш компьютер.Пошаговый запуск:Откройте IntelliJ IDEA на своем компьютере.Создайте новый проект: File ➔ New ➔ Project (выберите обычный Java Project).В левой панели проекта найдите синюю папку src. Нажмите по ней правой кнопкой мыши и выберите New ➔ Java Class. 

Назовите его строго Main (или FreaXExplorer, в зависимости от того, как назван public class в коде).
Скопируйте весь наш последний Java-код и вставьте его внутрь созданного класса, полностью заменив шаблонный текст.

Запуск: В самом верху IntelliJ IDEA нажмите на зеленый треугольник (кнопку Run) или используйте сочетание клавиш Shift + F10.

Результат: На вашем рабочем столе мгновенно откроется аккуратное ретро-окно 640x480 с мигающим зеленым люминофором. Клавиатура будет активна сразу — жмите F5 (пойдет симуляция дискеты со звуком мотора и писком ошибок) или вводите слово freax, чтобы Tux замахал крыльями без каких-либо задержек!
