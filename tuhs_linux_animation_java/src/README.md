# 💾 FreaXExplorer v1.91 — Ретро-терминал археологии Unix & Linux


Элегантное десктопное приложение на **Java Swing**, стилизованное под монохромный зеленый CRT-монитор (люминофор P1, IBM 5151). Программа имитирует низкоуровневую «археологическую» выгрузку исторических артефактов Unix/Linux из архивов **TUHS (The Unix Heritage Society)**.

---

## 🛠️ Основные фичи

* **Симуляция дискеты 3.5"**: Пошаговое чтение магнитных дорожек (CHS-адресация) с аналоговыми задержками и случайными сбоями секторов (CRC Error).
* **Электронный PC Speaker**: Интеграция звукового отклика (`Toolkit.beep()`) при сбоях чтения и подтверждении операций.
* **Исторические пасхалки**: Реальные комментарии Линуса Торвальдса из исходников ядер `v0.01-0.11` и хроники дебатов с Эндрю Таненбаумом.
* **Глобальный Key Dispatcher**: Железобетонный перехват горячих клавиш, полностью защищенный от блокировок текстового поля.

---

## 🎮 Управление (Горячие клавиши)

* **`[F5]`** — Запустить механическую эмуляцию чтения дискеты с архивом.
* **`[F8]`** — Открыть оригинальный веб-сайт сообщества **TUHS** в браузере.
* **`[Y / N]`** — Интерактивный ответ на запрос перенаправления в конце симуляции.
* **`f-r-e-a-x`** — Секретный пасхальный код. Наберите его посимвольно в любой момент, чтобы вызвать покадровую ASCII-анимацию «моргающего» пингвина **Tux** и узнать историю кодового имени ОС.

## 🚀 Запуск в GitHub Codespaces (в 1 клик)

Проект полностью оптимизирован для работы в облаке через **noVNC** и **Xvfb** (виртуальный фреймбуфер).

1. Откройте терминал внутри папки проекта.
2. Соберите и запустите проект одной командой:
   ```bash
   make
   ```
3. Перейдите на вкладку **Ports** (Порты) на нижней панели VS Code.
4. Найдите веб-порт **`6080`** и нажмите на иконку глобуса (**Open in Browser**).
5. В открывшейся вкладке нажмите большую синюю кнопку **Connect**.

## 💻 Локальный запуск (Windows / Linux / macOS)

Если вы запускаете код на домашнем компьютере, виртуальный графический сервер не нужен. Просто скомпилируйте файл напрямую:

```bash
javac Main.java
java Main
```
